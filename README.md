## 📌 Overview

**NexuxCart** is a modern full-stack inspired e-commerce web application designed to deliver a seamless online shopping experience. The platform focuses on performance, scalability, and user-friendly design while integrating modern frontend technologies and intelligent shopping workflows.

The application provides a clean and responsive shopping interface suitable for future expansion into a complete AI-enhanced commerce ecosystem.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sudarshan-118/NexuxCart.git
```

### 2️⃣ Navigate into the Project

```bash
cd NexuxCart
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

Now open:

```bash
http://localhost:3000
```
